# Docs

The `docs/` directory contains project documentation for contributors and users.

Expected documentation:

- `monorepo-structure.md`: repository structure and ownership guide.
- `cli.md`: command reference for `flutter_factory`.
- `bricks.md`: brick authoring guide and available brick list.
- `architecture.md`: generated app architecture conventions.
- `starter-template.md`: base starter template behavior.
- `contributing.md`: contribution workflow and review expectations.
- `assets/`: images, diagrams, screenshots, and the demo GIF.
